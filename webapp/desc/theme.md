# Theme Color Specification

## Theme Name

**Stage Neon**

## Theme Overview

This theme is inspired by **modern stage lighting and live performance environments**.
It uses a palette of **dark stage background + deep stage red + teal side lighting** to recreate the atmosphere of a theater stage.

The design emphasizes **visual focus and dramatic contrast**, similar to performers illuminated by stage lighting. The interface should feel like a **theater environment**, rather than a typical **AI or SaaS dashboard**.

The theme includes two modes:

- **Dark Mode — Night Stage**: simulates the atmosphere of a theater during a performance.
- **Light Mode — Stage Lights On**: represents the stage environment with lighting fully turned on.

Core palette structure:

**Dark stage environment + deep stage red spotlight + teal side lighting**

---

# Core Theme Colors (Shared Across Modes)

### Primary Color (Stage Spotlight)

**Stage Crimson**
`#973247`

A deep theatrical red inspired by **classic theater curtains and stage lighting**.
It should stay visually vivid under dark UI surfaces, but sit a little deeper than the earlier crimson to avoid a pinkish or overly saturated impression in compact controls.

Used for:

- Primary buttons
- Key interactions
- Highlighted UI elements
- Important actions

---

### Secondary Accent (Side Stage Lighting)

**Side Light Teal**
`#2F7A78`

A teal tone inspired by **stage side lighting and fill lights**.

Used for:

- Hover states
- Secondary actions
- Selection indicators
- Informational highlights

---

### Structural Color (Stage Floor)

**Stage Walnut**
`#3A2A22`

A deep brown tone inspired by **wooden theater stage floors**.

Used for:

- Structural UI sections
- Background panels
- Layout separation

# Design Principles

The theme follows a simple visual concept:

**Stage Environment + Lighting Focus**

- The interface acts like a **theater space**
- Key UI elements behave like **performers under a spotlight**
- The palette stays **minimal and dramatic**
- Avoid common **AI-style gradients or tech blues**

The final visual impression should evoke:

**Theater · Stage Performance · Spotlights · Live Shows**

rather than:

**AI tools · Tech dashboards · Generic SaaS interfaces**.
